\section{Basics}

Initilize repository:
\begin{verbatim}
  git init
\end{verbatim}

View status:
\begin{verbatim}
  git status
\end{verbatim}

Add to snapshot:
\begin{verbatim}
  git add
\end{verbatim}

Commit snapshot:
\begin{verbatim}
  git commit 
\end{verbatim}

View a repository's commit history:
\begin{verbatim}
  git commit 
\end{verbatim}

Define author name and email:
\begin{verbatim}
  git config user.name
  git config user.email
\end{verbatim}
